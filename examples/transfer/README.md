# Transfer

This is a complete example of an application. It uses `eosjs` to submit actions
to the blockchain.

## Getting started

To run the app in development mode:

```bash
npm install

npm run generate

npm run dev
```

To build the app:

```bash
npm install

npm run generate

npm run build
```

You can then open `public/index.html` in your browser to view the app.
